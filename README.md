# SkillCraft Technology - Data Analytics Internship Tasks

This repository contains all 4 tasks completed during my Data Analytics internship at SkillCraft Technology, covering Excel, Python (Pandas), and Power BI.

---

## 📅 Task 01: Excel Sales Dashboard
### 📌 Objective
Clean and analyze Superstore Sales Data using Microsoft Excel to build an interactive sales performance tracker.
* **Key Features:** Used **Pivot Tables** and **Pivot Charts** to analyze Total Sales, Profit by Category, and Sales Trends over time.
* **Formatting:** Applied **Conditional Formatting** to instantly highlight high and low-performing regions/products.

---

## 🐍 Task 02: Data Cleaning and Preparation (Python)
### 📌 Objective
Load a raw dataset into a Python environment using the **Pandas** library to make it analysis-ready.
* **Operations Performed:** * Identified and handled missing/null values.
  * Detected and dropped duplicate rows.
  * Converted mismatched data types (strings to proper date/numeric formats).
* **Output:** Exported the fully cleaned data into a new structured CSV file.

---

## 📊 Task 03: Power BI HR Employee Attrition Dashboard
### 📌 Objective
Build an interactive HR analytics dashboard to discover why employees are leaving the organization.
* **Key Features:** Added interactive slicers for Department, Age Group, and Business Travel.
* **Insights:** Discovered that the Research & Development department faced the highest attrition, and lower salary brackets directly correlated with higher employee turnover.

---

## 📈 Task 04: Marketing Campaign Business Insights (EDA)
### 📌 Objective
Perform Exploratory Data Analysis (EDA) on a marketing campaign dataset in Power BI and deliver data-driven budget recommendations.
* **Data Cleaning:** Successfully handled null entries in the Customer Income column.
* **Insights & ROI:** Analyzed funnel conversions from Campaign 1 to 5. Found that **Campaign 4** and the **Final Response** yielded the highest conversion rates, resulting in a recommendation to shift 20% more budget toward these channels.

---

## 🛠️ Tools & Technologies Used
* **Microsoft Excel** (Data Analysis, Pivot Charts, Conditional Formatting)
* **Python & Pandas** (Data Cleaning, Manipulation, ETL)
* **Power BI Desktop** (Data Visualization, DAX, Interactive Dashboards)
* **Power Query Editor** (Data Cleaning)
